SECTION TEXT
FOR: INPUT N + 0; ERROR 1 VARIVÁVEL N NÃO DECLARADA
    LOAD X + 0d2D ; ERROR 2 INTEIRO INVÁLIDO
    ADD X ; OK ACESSA POSIÇÃO 0
    ADD Y + 3 ; ERROR 4 SEG FAULT
    ADD X Y ; ERROR 5 ADD COM MAIS DE 1 PARAMETRO
    ADDI X ; ERROR 7 INSTRUÇÃO QUE NÃ OEXISTE
    DIV ZERO ; ERROR 8 DIVISÃO POR 0
    ADD 2 ; ERROR 9 NAO PODE SER IMEDIATO, TEM QUE SER LABEL
    LOAD X + Y ; ERROR 10 OFFSET TEM QUE SER LABEL
    STORE ZERO ; ERROR 11 ZERO EH CONSTRNATE
    ADD FOR ; ERROR 12 LABEL FOR NAO PODE SER SOMADA
    STOP
OUT:
    OUTPUT N ; ERROR 13 VARIVÁVEL N NÃO DECLARADA
    JUMP X ; ERROR 14 JUMP PRA LABEL QUE NAO É PULÁVEL
    JUMP FIM ; ERROR 15 LABEL NAO EXISTE
SECTION DATA

ERROR12: SPACE -1 ; NÃO PODE NUMERO NEGATIVO
X : SPACE 3
Y : SPACE 1
ZERO : CONST 0
